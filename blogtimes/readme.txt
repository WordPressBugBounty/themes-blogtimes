== Description ==

Blogtimes is a fast, clean, modern-looking Best Responsive News Magazine WordPress theme. The theme is fully widgetized, so users can manage the content by using easy to use widgets. Blogtimes is suitable for dynamic news, newspapers, magazine, publishers, blogs, editors, online and gaming magazines, newsportals,personal blogs, newspaper, publishing or review siteand any creative website. Blogtimes is SEO friendly, WPML,Gutenberg, translation and RTL ready. Live preview: https://demos.themeansar.com/blogus/blogtimes/ and documentation at https://docs.themeansar.com/docs/blogus/
